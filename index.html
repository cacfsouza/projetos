Conceitos importantes

(Voltar)

Clients

São os usuários, as pessoas que vão acessar o site ou blog e que estão em busca de algum conteúdo oferecido pelo autor.

Um site que ensina sobre idiomas, atrairá pessoas interessadas neste conteúdo. Estas pessoas são os clients, ou em português clientes.

Navegadores

Os navegadores são os programas usados para acessar os sites da internet. Os mais conhecidos são:

  • Google
  • Firefox
  • Edge
  • Safari

Alguns navegadores já vem instalados no seu computador. Se o seu sistema for Windows o navegador padão é Edge.

No caso do iOS, sistema da Apple, o navegador padrão é o Safari.

Servers

Para entender de maneira bem simples, um server - servidor - é um computador ou software mais potente que um computador comum, ou o que um computador comum pode oferecer.

Ele possui processadores, além de bancos de memórias, portas de comunnicação, diferentes softwares e sistemas de armazenamento de dados, muito potentes.

O servidor é um computadorou software central que forma uma arquitetura"cliente-servidor".Esta arquitetura conecta o servidor com outros dispositivos, como:

  • computadores
  • tablets
  • smartphones

Hospedagem

Hospedagem nada mais é que um serviço que possibilita pessoas ou empresas com sistemas online a guardar páginas, arquivos, informações, etc, ou qualquer conteúdo acessível online.

Ou seja, a hospedagem é um espaço dentro de um servidor onde os arquivos e páginas de um site ficam guardados.

Existem diversos tipos de hospedagem de sites, dentre eles:

  • Hospedagem grátis
  • Hospedagem compartilhada
  • Servidor dedicado
  • Nuvem

Mas qual a diferença entre Servidor e Hospedagem?

A hospedagemé o serviço que guarda o conteúdo do site e o torna acessível aos usuários.

O servidor, por outro lado, consiste no local onde esse conteúdo será guardado, como as imagens, textos, vídeos, música e etc.

Conceito

HTMLnão é uma linguagem de programação. HTMLé um arquivo de texto, tipo .doc mas com textos delimitados, fazendo com o que os navegadores entendam/interpretem os comandos, e assim apareça no seu navegador, celular ou tablet.

Este texto delimitado é, melhor dizendo, delimitadores tem um nome e uma função, assim como um comportamento específico.

Veja abaixo a estrutura básica do HTML

Estrutura Básica

Para iniciar com o HTML você precisa de um programa para escrever o código. Sugerimos utilizar o Visual Studio Code.

Ao iniciar um novo projeto, você vai criar uma página chamada "index.html", e então aparecerá a estrutura básica para iniciar o seu novo projeto.

Aparecerão as tags:

  • html e /html, que marcam o início e fim da página sendo criada.
  • head e /head, que é o título da página. Como o dessa aqui, que você pode ler lá em cima - HTML na Prática!
  • body e /body, o corpo do projeto. Justamente onde serão digitados todos os comandos, linhas, texto e afins, da página que estará criando.

É dentro das tags body e /body que você irá criar toda a sua página e ver o resultado de tudo que está fazendo.

Tags

As Tags são usadas para informar o navegador a estrutura do site. Quando escrevermos o código HTML, as Tags serão interpretadas pelo navegador produzindo a estrutura e conteúdo da página.

A principal característica das tags é estar sempre dentro dos sinais chevron, ou seja "<" - maior que ou ">" - menor que.

As tags também podem ter um ou maisatributos, que modificarão a estrutura da tag, ou sua funcionalidade.

Os atributos são usados para personalizar as tags, porém a maioria das tags possuem os seus próprios atributos. Que serão vistos em outro momento.

Veja abaixo alguns exemplos mais comuns de tags:

  1. Tags de Título
      2. Usadas para definir título, variando entre 6 níveis hierárquicos:do h1 ao h6 - maior ao menor tamanho de título.
  2. Tags de Link
      3. Esta tag é utilizada para fazer a ligação entre um documento e outro, sendo ele de uma mesma página ou de outro domínio. Para esta tag usamos a href="dio.me"
  3. Tags de Listas
      4. As tags de listas são muito utilizadas quando queremos criar listas numeradas ou por bullet points. Para criar as listas usamos: ol e /ol para listas numeradas; e ul e /ul para listas em bullet points.
